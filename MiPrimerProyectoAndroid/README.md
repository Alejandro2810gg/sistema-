# Mi Primer Proyecto Android

Proyecto Android de ejemplo que demuestra:

- ✅ Soporte multi-idioma (Español, Inglés, Francés, Alemán)
- ✅ Soporte para distintos tamaños de pantalla (teléfono, tablet, TV)
- ✅ Soporte para orientación vertical y horizontal
- ✅ Imagen de fondo con nine-patch
- ✅ Pantalla de bienvenida con botón de entrada

## Capturas

La app muestra una pantalla de bienvenida con:
- Texto "Bienvenido" adaptado al idioma del dispositivo
- Botón "Entrar" / "Enter" / "Entrer" / "Betreten"
- Imagen de fondo (mascota Android sobre nubes)

## Idiomas soportados

| Idioma | Texto |
|--------|-------|
| Español (default) | Bienvenido / Entrar |
| Inglés | Welcome / Enter |
| Francés | Bienvenue / Entrer |
| Alemán | Willkommen / Betreten |

## Cómo ejecutar

1. Clona el repositorio: `git clone https://github.com/TU_USUARIO/MiPrimerProyectoAndroid.git`
2. Abre el proyecto en **Android Studio**
3. Ejecuta en emulador o dispositivo físico

## Requisitos

- Android Studio Hedgehog o superior
- SDK mínimo: API 21 (Android 5.0)
- SDK objetivo: API 34 (Android 14)
